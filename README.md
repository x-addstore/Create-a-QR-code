# ระบบสร้าง QR CODE ฟรี พร้อม ซอสโค้ด โค้ดเพื่อการศึกษาเท่านั้น ขออนุญาติเจ้าของ API

นี่คือคู่มือการใช้งาน **goqr.me API** (หรือ `api.qrserver.com`) ซึ่งเป็นหนึ่งใน API ยอดนิยมสำหรับการสร้าง QR Code ที่ใช้งานได้ฟรี ไม่จำเป็นต้องลงทะเบียนเพื่อขอ API Key และไม่มีข้อจำกัดที่ซับซ้อนครับ เหมาะสำหรับนักพัฒนาที่ต้องการระบบที่เสถียรและทำงานได้รวดเร็ว

---

## 📌 ข้อมูลพื้นฐานของ API (Base URL)

ทุกการส่งคำร้องขอ (Request) เพื่อสร้าง QR Code จะต้องส่งผ่าน URL หลักนี้:

> **`[https://api.qrserver.com/v1/create-qr-code/](https://api.qrserver.com/v1/create-qr-code/)`**
> * **Method:** `GET` หรือ `POST` (แนะนำใช้ `GET` เพื่อความง่ายและรวดเร็วในการแสดงผลบนหน้าเว็บ)

---

## 🛠️ พารามิเตอร์ที่ใช้งานบ่อย (Parameters)

คุณสามารถกำหนดค่าต่างๆ ของ QR Code ผ่านการต่อ Query String ด้านหลัง URL หลักได้ดังนี้ครับ:

| พารามิเตอร์ | ประเภทข้อมูล | บังคับใช้งาน? | รายละเอียดและการตั้งค่า |
| --- | --- | --- | --- |
| **`data`** | String (Text/URL) | **ใช่ (Required)** | ข้อความ หรือ URL ที่ต้องการแปลงเป็น QR Code **(สำคัญ: ต้องทำการ `URL-encode` ข้อมูลก่อนส่งเสมอ)** |
| **`size`** | String (กว้างxสูง) | ไม่บังคับ (Default: 150x150) | ขนาดของรูปภาพเป็นพิกเซล (px) เช่น `300x300` หรือ `500x500` (ขนาดสูงสุดที่แนะนำคือ `1000x1000`) |
| **`format`** | String (ชนิดไฟล์) | ไม่บังคับ (Default: png) | รูปแบบไฟล์ที่ต้องการให้ตอบกลับ ได้แก่ `png`, `gif`, `jpeg`, `svg` (แนะนำใช้ SVG หากต้องการขยายรูปภาพแล้วภาพไม่แตก) |
| **`color`** | String (Hex Code) | ไม่บังคับ (Default: 000000) | สีของตัว QR Code โดยระบุเป็นรหัสสี Hex (ไม่รวมเครื่องหมาย #) เช่น `0f172a` |
| **`bgcolor`** | String (Hex Code) | ไม่บังคับ (Default: ffffff) | สีพื้นหลังของ QR Code ระบุเป็นรหัสสี Hex เช่น `f1f5f9` |
| **`ecc`** | String (L | M | Q |
| **`margin`** | Integer (ตัวเลข) | ไม่บังคับ (Default: 1) | ขนาดขอบสีขาวรอบๆ QR Code (หน่วยเป็นจำนวนบล็อกพิกเซล) ตั้งค่าได้ตั้งแต่ `0` ถึง `50` |

## 💡 ตัวอย่างการเรียกใช้ (Examples)

### 1. การใช้งานแบบง่ายที่สุด (Default Settings)

แปลงลิงก์ `[https://google.com](https://google.com)` เป็น QR Code ขนาดเริ่มต้น:

```http
https://api.qrserver.com/v1/create-qr-code/?data=https%3A%2F%2Fgoogle.com

```
